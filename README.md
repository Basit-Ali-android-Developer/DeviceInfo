# Device Info App

An Android application to display detailed information about a device, built entirely in **Kotlin**.

---

## Features

- **Full Device Information**: RAM, memory, processor, installed apps, battery, display, networks.
- **Separate Pages**: Each category of device info has its own dedicated screen.
- **Themes**: Supports **Dark**, **Light**, and **Custom** themes.
- **Multilingual Support**: Available in **English**, **Urdu**, and **Arabic**.
- **Offline App**: No backend or external storage required â€” works entirely on the device.

---

## Tech Stack

- Language: Kotlin
- IDE: Android Studio
- Architecture: Standard Android structure
- Themes: Day/Night + Custom theme support
- Localization: English, Urdu, Arabic
